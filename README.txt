Framework openMairie
====================

Le framework openMairie est une librairie PHP qui permet de créer une
application très rapidement en générant, à partir du modèle de données, des
formulaires, des listings, des éditions PDF, des affichages sur carte et bien
plus encore.

Toutes les informations sur http://www.openmairie.org/framework


Documentation
-------------

Toute la documentation pour l'installation, l'utilisation et la prise en main
du framework dans le guide du développeur en ligne :
http://docs.openmairie.org/?project=framework&format=html
